# 설치법

.obsidian\plugins\ 폴더 안에
obsidian-hwpx-writer라는 이름으로 폴더를 만들고 위 깃허브 파일을 복사하시면 됩니다.

처음으로 인스톨하면 나오는 설치 가이드가 3단계로 뜹니다. 
1) 파이썬 설치 확인
2) pandoc 설치 확인
3) pypandoc hwpx 설치 확인

하라는 대로만 하면 이제 옵시디언에서 한글 내보내기 및 편집 및 내 템플릿 지정 및 툴바 기능까지 몽땅 사용하실 수 있습니다.

# 특정 서식 템플릿 적용하는 법

한글 내보내기는 테마 상관없이 기본적으로 blank.hwpx를 정확하게 따라가게 되어있습니다

새 서식 템플릿을 만드시려면 blank.hwpx를 따로 복사하시고 F6을 눌러서
Heading 1-6과 바탕글에 해당하는 거를 자간 줄간격 들여쓰기 포인트 등을 미세조정하신후 파일 저장해서
그걸 옵시디언에서 새 템플릿으로 등록해놓으면 즉시 보입니다.
